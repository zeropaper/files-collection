# FilesCollection

A [Ampersand.js](://ampersandjs.com) set of classes to help files and directories handling.

## Documentation

Can be generated using npm `run-script docs` and will be written in `DOCUMENTATION.md`.

## Test

Can be runned usin `npm test`.
